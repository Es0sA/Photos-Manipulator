# Privacy Policy: Photos Manipulator Bot

_Last updated: 2026-07-12_

## What this bot does with your data

When you send a photo to this bot:

1. The file is downloaded to a server running this bot, via a self-hosted
   Telegram Bot API server under the bot operator's control. No image is
   sent to any third-party AI or storage service.
2. All processing (upscaling, background removal, restoration, colorization,
   format conversion) happens locally on that server using open-source
   models running on the server's own hardware.
3. The result is sent back to you as a Telegram message.
4. The original file and any generated output are deleted immediately
   after they're sent (or if processing fails). Nothing is stored, logged,
   or retained afterward.

## What isn't collected

- No photo is kept after your request is processed.
- No data is sold, shared, or used for advertising.
- No data is used to train any model.

## Third parties involved

- **Telegram** delivers your message to the bot and the bot's replies back
  to you. Telegram's own [Privacy Policy](https://telegram.org/privacy)
  governs how Telegram itself handles your messages and account data.

## Server logs

Basic operational logs (e.g. errors, request timing) may be kept briefly for
debugging the bot itself. These logs do not include image content.

## Contact

Questions about this policy or how the bot handles your data can be raised
via the [GitHub repository](https://github.com/Es0sA/Photos-Manipulator).
